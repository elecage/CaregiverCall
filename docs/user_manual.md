# User Manual

이 문서는 CaregiverCall 기기의 실제 사용 방법을 설명합니다.

## 기기 목적

CaregiverCall은 집 안에 있는 사용자가 버튼을 눌러 외부 보호자에게 도움 요청을 보내고, 보호자가 확인했음을 다시 안내받기 위한 보조기기입니다.

## 기본 사용 방법

1. 기기가 전원에 연결되어 있는지 확인합니다.
2. 도움이 필요하면 큰 호출 버튼을 한 번 누릅니다.
3. 기기의 LED, 부저, 진동 등 로컬 피드백이 있다면 호출이 접수되었는지 확인합니다.
4. 보호자가 스마트폰에서 호출을 확인하면 Google Nest Mini가 확인 안내를 말합니다.
5. 확인 안내 예시는 다음과 같습니다.

```text
보호자가 호출을 확인했습니다.
```

## 상태 안내

아직 최종 하드웨어 피드백 방식은 확정되지 않았습니다. 아래는 권장 상태 표시입니다.

| 상태 | 권장 피드백 |
| --- | --- |
| 대기 중 | 외부 LED 꺼짐 또는 약한 점등 |
| 호출 전송 중 | 외부 LED 천천히 점멸 |
| 호출 전송 실패 | 외부 LED 빠르게 점멸 또는 진동 패턴 |
| 보호자 확인 완료 | 외부 LED 일정 시간 점등, Nest Mini 음성 안내 |

## 보호자 사용 방법

1. 스마트폰에서 호출 알림을 확인합니다.
2. 사용자가 도움을 요청했다는 내용을 확인합니다.
3. 확인 버튼을 누릅니다.
4. 필요한 경우 사용자에게 전화하거나 직접 방문합니다.

## 주의사항

- 이 기기는 보호자 호출을 돕는 보조기기입니다.
- 네트워크 장애, 스마트폰 알림 차단, Nest Mini 오프라인 상태에서는 알림이나 음성 안내가 지연되거나 실패할 수 있습니다.
- 긴급 상황에서는 전화, 문자, 응급 호출 등 별도 연락 수단을 함께 사용하세요.
- 기기를 물에 젖게 하거나 불안정한 전원에 연결하지 마세요.

## 정기 확인

- 하루에 한 번 이상 버튼 시험 호출을 해보는 것을 권장합니다.
- Nest Mini 음량이 충분한지 확인합니다.
- 보호자 스마트폰의 알림 권한이 켜져 있는지 확인합니다.
- 기기 전원 케이블이 빠져 있지 않은지 확인합니다.
